Guía de Instalación

Requisitos

  • Python 3.x
  • pip
  • Git

Pasos

  1. Clona el repositorio: bash git clone https://github.com/tu_usuario/Documentacion_Proyecto.git

  2. Navega al directorio: bash cd Documentacion_Proyecto

  3. Instala dependencias: bash pip install mkdocs

  4. Inicia el servidor local: bash mkdocs serve

  5. Verifica en: http://127.0.0.1:8000
